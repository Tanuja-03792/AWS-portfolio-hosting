# Portfolio Website Hosting using AWS EC2

## Project Title
Harnessing EC2, VPC & MobaXterm for Cutting-Edge Entity Hosting

## Technologies Used
- AWS EC2
- Amazon Linux
- Apache Server
- MobaXterm
- HTML
- CSS
- JavaScript

## Project Flow
1. Created AWS account
2. Launched EC2 instance
3. Generated key pair
4. Configured SSH, HTTP and HTTPS
5. Connected EC2 using MobaXterm
6. Installed Apache server
7. Uploaded portfolio files
8. Hosted website on Public IP
